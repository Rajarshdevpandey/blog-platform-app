# Task 4: Blog Platform with Comments

A full-stack Blog Application developed as part of the **Thiranex Web Development Internship**. This platform allows users to register, log in, create blog posts, edit or delete their own posts, and engage with other users through a real-time comment section.

## 🚀 Features
- **User Authentication:** Secure Login and Registration system.
- **CRUD Operations:** Users can Create, Read, Update (Edit), and Delete blog posts.
- **Interactive Comment Section:** Allows users to post public comments under any blog article.
- **Full-Stack Architecture:** Powered by a responsive frontend and integrated with backend RESTful APIs and a database structure.

## 🛠️ Tech Stack Used
- **Frontend:** HTML5, CSS3 (Responsive Grid Design), JavaScript (ES6)
- **Backend Architecture:** Node.js, Express.js (RESTful APIs Setup)
- **Database Logs Structure:** MongoDB Schema Models
